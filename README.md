# SSL_Med

Dataset: [Left Atrium](https://github.com/yulequan/UA-MT/tree/master/data/2018LA_Seg_Training%20Set), [Pancreas](https://wiki.cancerimagingarchive.net/display/Public/Pancreas-CT)
